# Hermes HUD 看板 — 一键安装包

> 为朋友提供的快速安装方案

## 安装步骤

### 第1步：安装看板

**双击 `install-windows.bat`**，脚本会自动完成：

✓ 检查你的电脑环境（Node.js、Python、Hermes）
✓ 下载 hermes-hudui
✓ 安装 Python 后端
✓ 修补 Windows 兼容问题
✓ 安装 5 套自定义皮肤
✓ 构建前端界面
✓ 桌面生成「启动Hermes看板.bat」

### 第2步：启动看板

**双击桌面「启动Hermes看板.bat」**

浏览器打开 http://localhost:3001

按键盘 `t` 键可以切换皮肤（共10套）。

### 第3步（可选）：设置开机自启

**双击 `setup-autostart.bat`**

以后电脑开机，看板自动在后台运行，不用手动启动。

## 前提条件

- Hermes Agent 已安装并运行过
- Python 3.11+（如果没有，脚本会提示）
- Node.js 18+（如果没有，脚本会提示）
