# 🏔️ Hermes Agent 主题皮肤：青绿国风（Cyan Ink）

> **设计基因**：深墨绿 `#004B45` + 纯白 `#FFFFFF` + 粗体几何字
> **设计概念**：提取中国传统"青绿山水"与"景泰蓝"的配色逻辑，将极简几何与东方美学融合，打造数字化的"文人案头"与"宫廷御览"之间的现代平衡。

---

## 🎨 色彩系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 石青绿 | `#004B45` | 主背景（品牌原色即石青近亲） |
| 宣纸白 | `#F7F5F0` | 主文字（暖白，如熟宣） |
| 朱砂红 | `#C9372C` | 印章、重点、完成状态 |
| 花青蓝 | `#1E6E6E` | 次级文字、水墨层次 |
| 泥金 | `#D4AF37` | 边框、装饰纹样、徽章 |

---

## 🔤 字体风格

- **标题字体**：`方正清刻本悦宋` 或 `思源宋体`
- **正文字体**：`霞鹜文楷` 或 `Noto Sans SC`
- **排版逻辑**："楷宋混排"的古典阅读节奏
- **特殊排版**：竖排文字用于装饰性标签（`writing-mode: vertical-rl`）

---

## 🖼️ 视觉元素

1. **底纹纹理**：背景叠加极淡的"水波纹"或"云纹" SVG 纹理（`opacity: 0.04`，泥金色）
2. **印章 Logo**："Mn" Logo 纯白，外围增加圆形"印章"底托：
   - 方案 A：朱砂红圆底 + 白字
   - 方案 B：纯白字 + 泥金细圆框
3. **回纹角花**：卡片边角采用极小 SVG 回纹装饰角花（8×8px），不破坏极简
4. **云纹分割线**：分割线使用"山形"或"云纹"虚线（2px 泥金，间隔 4px）
5. **朱砂批注**：引用与完成状态使用朱砂红"批注"样式，右侧竖排小字"已阅"或"批"
6. **水墨渐变**：次级背景使用花青蓝到石青绿的微妙渐变（`linear-gradient(180deg, #004B45, #1E6E6E)`）

---

## 💡 设计师备注

- 泥金色值 `#D4AF37` 在不同屏幕上色差较大，建议提供 SVG 矢量装饰替代纯色
- 竖排文字仅用于装饰，核心信息保持横排以确保可读性
- 云纹/水波纹背景需严格控制透明度（≤ 4%），避免干扰正文阅读
- **推荐作为品牌官方默认主题**——最贴合原图气质，兼具专业感与文化辨识度
