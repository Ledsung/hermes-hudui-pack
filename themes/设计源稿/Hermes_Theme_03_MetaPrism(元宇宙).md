# 🌌 Hermes Agent 主题皮肤：元宇宙棱镜（Meta Prism）

> **设计基因**：深墨绿 `#004B45` + 纯白 `#FFFFFF` + 粗体几何字
> **设计概念**：保留品牌绿的暗色基底，但引入元宇宙的故障艺术（Glitch）、全息投影与空间折射，让界面具有"数字孪生"的虚实叠加感。

---

## 🎨 色彩系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 矩阵绿 | `#004B45` | 主背景（保留品牌原色） |
| 全息白 | `#FFFFFF` | 主文字 |
| 赛博紫 | `#B829DD` | 全息投影、侧边光 |
| 荧光粉 | `#FF2A6D` | 警报、异常状态、NFT 标记 |
| 虚空黑 | `#001211` | 对话框阴影、深层背景 |

---

## 🔤 字体风格

- **首选字体**：`Clash Display` 或 `Space Grotesk`
- **字重**：Bold / Extrabold
- **特征**：粗体且带有轻微的几何切割感
- **特效**：字母边缘可做 1px 的 RGB 分离（Glitch 效果）

---

## 🖼️ 视觉元素

1. **全息底噪**：背景添加缓慢流动的噪点纹理：`opacity: 0.03`，模拟全息底噪
2. **Glitch Logo**："Mn" Logo 采用三层错位叠加：
   - 底层：赛博紫（左移 2px）
   - 中层：荧光粉（右移 2px）
   - 顶层：纯白
3. **毛玻璃卡片**：`backdrop-filter: blur(12px)` + 半透明白边框（`rgba(255,255,255,0.2)`）
4. **渐变分割线**：1px 渐变线（紫 → 粉 → 青），替代纯白线
5. **悬浮粒子**：背景增加缓慢上升的微小光点（`#00E5C9`，`opacity: 0.3`）

## 💡 设计师备注

- Glitch 效果建议使用 CSS `text-shadow` 多层叠加实现，避免使用图片以保证性能
- 毛玻璃效果在低端设备上需降级为半透明纯色背景（`fallback` 机制）
- 荧光粉与赛博紫在深色背景上对比度需人工校验，确保可读性
