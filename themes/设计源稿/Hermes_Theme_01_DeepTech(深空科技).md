# 🚀 Hermes Agent 主题皮肤：深空科技（Deep Tech）

> **设计基因**：深墨绿 `#004B45` + 纯白 `#FFFFFF` + 粗体几何字
> **设计概念**：将品牌绿视为"深海/深空"的暗色基底，融入精密科技的光效与数据流质感，打造 NASA 地面指挥中心般的专业操控界面。

---

## 🎨 色彩系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 深空绿 | `#003B36` | 主背景（比品牌色深10%） |
| 科技白 | `#E8F4F2` | 主文字与图标 |
| 数据青 | `#00E5C9` | 交互高亮、光标、数据流 |
| 网格灰 | `#1A4A44` | 面板分割、次级背景 |

---

## 🔤 字体风格

- **首选字体**：`JetBrains Mono` 或 `SF Mono`
- **字重**：Bold / Heavy
- **特征**：等宽粗体，字母间距略紧（`-0.02em`）
- **氛围**：强化代码与终端的精密感

---

## 🖼️ 视觉元素

1. **背景网格**：叠加 5% 透明度的 45° 网格线（1px 细线，颜色 `#00E5C9`）
2. **交互光晕**：交互元素采用 1px 科技青边框，`hover` 时触发霓虹光晕：
   ```css
   box-shadow: 0 0 8px #00E5C9;
   ```
3. **Logo 动效**：头像/Logo 区域保留纯白粗体 "Mn"，增加扫描线（scanline）微动效
4. **卡片风格**：无边框或 1px 网格灰实线，直角或 2px 微圆角

---

## 🧠 界面氛围

冷静、理性、极客、控制台感、零冗余信息

---

## 🎯 适用场景

代码生成、数据分析、技术调试、开发助手模式

---

## 💡 设计师备注

- 建议基于 CSS 变量（`var(--bg-primary)`）构建，确保轻量化与响应式
- 扫描线动效可使用 CSS `linear-gradient` 背景 + `animation` 实现，避免性能开销
- 数据青色值在暗光环境下需通过 WCAG 对比度检测（目标 AAA 级）
